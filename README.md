# Nusrat Jahan | Nutritionist Portfolio

A high-performance, responsive portfolio website for **Nusrat Jahan**, a professional Nutritionist and Food & Nutrition specialist. This project features a modern aesthetic with smooth 3D animations, optimized performance, and SEO readiness.

✨ **Live Demo:** [https://nusratjnisha.netlify.app/](https://nusratjnisha.netlify.app/)

## 🚀 Key Features

- **Modern UI/UX**: Professional health-themed aesthetic using Poppins and Playfair Display typography.
- **Premium Interactivity**:
  - **3D Tilt Effects**: Dynamic card effects on specializations and certifications.
  - **Flip Cards**: 3D rotation on project cards for detailed views.
  - **Particle Engine**: Subtle floating background particles for depth.
  - **Magnetic Links**: Interactive social media icons with magnetic hover effects.
- **Performance Optimized**:
  - GPU-accelerated animations using `will-change`.
  - Smooth 3D effects optimized with `requestAnimationFrame`.
  - Lazy loading for images and scroll-reveal triggers.
- **SEO & Search Ready**:
  - **JSON-LD Schema**: Integrated `Nutritionist` markup for rich search snippets.
  - Open Graph tags for professional social media sharing.
- **Fully Responsive**: Tailored experiences for Mobile, Tablet, and Desktop.
- **Easter Egg**: Try the Konami Code for a celebration! (↑ ↑ ↓ ↓ ← → ← → B A)
