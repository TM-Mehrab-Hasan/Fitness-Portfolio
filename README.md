# Fitness/Nutritionist - Portfolio Website

A modern, responsive portfolio website for Nusrat Jahan, a dedicated Nutritionist and Food & Nutrition professional.

## Features

- **Modern Design**: Clean and professional layout with health-themed color scheme (greens)
- **3D Animations**: Interactive 3D card effects, floating icons, and parallax scrolling
- **Animated Elements**: 
  - Typing text effect in hero section
  - Floating particles background
  - Hover animations on buttons and cards
  - Ripple effects on button clicks
  - Magnetic social link hover effects
  - Flip animations on project cards
  - Smooth scroll animations
- **Responsive**: Fully responsive design for all devices
- **Interactive Timeline**: Experience section with animated timeline
- **Contact Form**: Email integration with mailto link

## Files Structure

1. **Add Profile Image**: Replace `profile.jpg` with Nusrat's actual photo (square image works best)

2. **Customize Links**: Update social media links in the HTML file if needed

## Free Hosting Options

### Option 1: GitHub Pages (Recommended)

1. Create a GitHub account at https://github.com
2. Create a new repository named `nusrat-portfolio` or `username.github.io`
3. Upload all files to the repository
4. Go to Settings > Pages
5. Select "main" branch and click Save
6. Your site will be live at `https://username.github.io/nusrat-portfolio`

### Option 2: Netlify

1. Go to https://netlify.com
2. Sign up with GitHub or email
3. Click "Add new site" > "Deploy manually"
4. Drag and drop the portfolio folder
5. Get your free .netlify.app URL

### Option 3: Vercel

1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "Add New Project"
4. Import or upload your files
5. Get your free .vercel.app URL

### Option 4: Render

1. Go to https://render.com
2. Create account and add a new Static Site
3. Connect your GitHub repo or upload files
4. Deploy and get your free URL

## Customization

### Colors
To change the color scheme, edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #4CAF50;       /* Main green */
    --primary-dark: #388E3C;        /* Darker green */
    --primary-light: #81C784;       /* Light green */
    ...
}
```

### Fonts

- **Playfair Display** - Headings
